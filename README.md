**Find your love... S2..**

→ 취미 : 영화, 쇼핑, 독서, 요리, 맛집탐방, ...
→ 성격 : 다정함, 부유함, 재밌음(유머러스), 활발함, 건강함, ... 
상수로 지정하고 리스트를 만들기

기능
회원가입, 게시글, 매치

회원 가입을 시작합니다.
